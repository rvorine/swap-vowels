# Swap Vowels - Python Implementation

## Description
This is a Python implementation of the Swap Vowels algorithm using the two-pointer technique.

## How to Run

```bash
python swap_vowels.py
```

Or with Python 3:
```bash
python3 swap_vowels.py
```

## Code Structure
- `swap_vowels(s: str)`: Main function that swaps vowels in a string
- `is_vowel(c: str)`: Helper function to check if a character is a vowel

## Requirements
- Python 3.6 or higher

## Example Output
```
Input:  "hello"
Output: "holle"

Input:  "leetcode"
Output: "leotcede"
```

## Usage as Module
You can also import this as a module:
```python
from swap_vowels import swap_vowels

result = swap_vowels("hello")
print(result)  # Output: "holle"
```
