# Swap Vowels - Java Implementation

## Description
This is a Java implementation of the Swap Vowels algorithm using the two-pointer technique.

## How to Run

### Compile:
```bash
javac SwapVowels.java
```

### Execute:
```bash
java SwapVowels
```

## Code Structure
- `swapVowels(String str)`: Main method that swaps vowels in a string
- `isVowel(char c)`: Helper method to check if a character is a vowel

## Requirements
- Java 8 or higher
- JDK installed

## Example Output
```
Input:  "hello"
Output: "holle"

Input:  "leetcode"
Output: "leotcede"
```
